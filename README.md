

# Link Tracker
LinterBot - проект отслеживания обновлений по ссылкам. Состоит из двух REST-сервисов: telegram-bot и scrapper. Telegram-bot принимает вводимые пользователем URL-ы и по HTTP передает их второму сервису - Scrapper-у, который обращается по API к соответствующим сервисам для анализа возможных обновлений и уведомляет о них сервис telegram-bot, который в свою очередь передает ее пользователю (Spring Boot, REST, Swagger)
